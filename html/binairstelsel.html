Binair stelsel

Het binaire stelsel, oftewel het tweetallige stelsel, wordt geteld met enkel de nummers 0 en 1. Dit is mogelijk, doordat iedere positie een macht is van 2. Binaire getallen worden gelezen van rechts naar links. Als je vanaf de rechterkant begint met tellen, wordt het nummer op de eerste positie vermenigvuldigd met 20. Het nummer op de tweede positie wordt vermenigvuldigd met 21. De getallen die uit de posities komen, tel je dus bij elkaar op. De code 01 is dus gelijk aan 1, de code 10 is gelijk aan 2 en de code 11 is gelijk aan 3. Dit stelsel wordt hoofdzakelijk gebruikt in computers, waarin de 0 en 1 aangeven of bijvoorbeeld een schakelaar aan moet staan. Eén zo’n waarde, dus één 0 of één 1, noem je een bit. Dit is de kleinste eenheid van informatie. Een rij van 8 bits noem je een byte. Voor grotere aantallen bytes worden de SI-voorvoegsels vaak toegepast. Daar komen namen zoals kilobyte, gigabyte en terabyte dus ook vandaan.

ASCII-tabel

ASCII (American Standard Code for Information Interchange) is een systeem binnen het binair stelsel waarmee computers letters, cijfers en leestekens kunnen vertonen. De code is afleesbaar, maar het is niet zo simpel als de letters die we met ons toetsenbord typen. Het zijn namelijk opdrachten aan uitvoerapparaten of geven informatie over gegevens die verstuurd worden. De code van de meeste leestekens begint met 010, gevolgd door de nummers gebonden aan een teken. De code van cijfers begint met 011, gevolgd door het gewenste cijfer in binaire code. Hoofdletters beginnen met 10, kleine letters beginnen met 11 en alle letters staan op alfabetische volgorde. ‘Binair!’ schrijf je bijvoorbeeld als '1000010 1101001 1101110 1100001 1101001 1110010 0100001'.

Hieronder vind je een klein spelletje, genaamd Binary Bonanza, waarmee je het tellen met het binaire stelsel kunt oefenen.
